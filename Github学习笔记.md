# 我的学习笔记

## 一、我眼中的geek

* 热爱与探索

​        极客往往对科技、计算机、数字产品等特定领域怀有极其浓厚的兴趣和热爱，他们会花大量时间用于计算机技术探索，去寻找计算机的各种可能性

* 创新与合作

​        极客会尝试各种新事物，突破传统思维的禁锢，追求技术上的突破，他们也会相互帮助，合作解决问题，互相分享知识（比如开源软件）

* 追求完美

  极客对自己有极高的要求，力求产品或技术的完美，平时写代码也会力求精简、直观。

## 二、关于github

​        我觉得GitHub就像是一个巨大的数据库，我可以上传自己写的代码，也可以阅读并使用他人上传的开源内容（可以白嫖好多软件和各种资源😀）

* fork：可以把自己感兴趣的项目复制下来，并在不影响原项目的情况下自由开发
* pull request：在fork的基础上，当修改了一个项目时，可以通过pull request把修改后的内容发给原作者，如果原作者同意，修改后的内容会被并入原项目中。
* 使用问题：在使用GitHub时我没有遇到进不去的情况，进不去可能是因为网络不稳定、DNS解析异常或者防火墙拦截，可以通过使用加速器或VPN以及镜像软件来解决问题。

## 三、关于markdown

​        markdown是一种轻量级标记语言，他的语法简洁，高效，写出的内容整齐美观，可以用于记笔记或网站生成，同时他的格式便于转换，使用起来很方便。

## 四、关于文档的整洁与规范

* 格式方面：使用了markdown编写。
* 命名方面：命名为“我的学习笔记”，简单明了。
* 内容方面：使用一、二级标题区分内容，使逻辑清晰，内容简洁，避免废话。
* 排版方面：正文字体保持一致，段落格式清晰。
